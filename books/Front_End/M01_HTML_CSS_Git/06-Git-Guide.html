How to Create a New Repository via Command Line

Follow these steps to create and manage a new Git repository for your web project.

Step 1: Navigate to Your Project Directory

cd path/to/your/directory

Step 2: Create a New Project Folder


mkdir my-web-project
cd my-web-project

Step 3: Initialize a Git Repository

git init

Step 4: Create an HTML File

touch index.html

Now, add the following basic HTML structure inside the index.html file:


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>My Web Project</title>
</head>
<body>
    <h1>Welcome to My Web Project</h1>
    <p>This is a sample project.</p>
</body>
</html>

Step 5: Stage the Changes

git add .

Step 6: Commit the Changes

git commit -m "Initial commit with index.html"

Step 7: Create a New Repository on GitHub

Create a new repository on GitHub, then copy the repository URL.

Step 8: Link the Local Repository with GitHub

git remote add origin https://github.com/your-username/my-web-project.git

Step 9: Push Your Changes to GitHub

git push -u origin main or master

Step 10: Final Professional Touch

Create a .gitignore and README.md file:


touch .gitignore
touch README.md

Add the following content to the README.md:


# My Web Project

This is a simple web project demonstrating HTML structure and how to create and manage a repository using Git and GitHub.
